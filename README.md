# 競馬予想補助ツール
netkeiba.comをスクレイピングして競馬予想に役立つ情報を出力します。


# できること

## 展開予想
出走馬の脚質を独自の方法で分析して表示します
<img width="473" alt="スクリーンショット 2023-04-20 0 28 02" src="https://user-images.githubusercontent.com/74707967/233124885-f121addb-274a-4229-9b5d-3e14a1728cb4.png">

## オッズ
オッズと脚質、過去の出走タイムをDataFrame型で表示します
<img width="779" alt="スクリーンショット 2023-04-20 0 28 41" src="https://user-images.githubusercontent.com/74707967/233125072-093c5379-6ff4-4959-a4e8-1229f06a3cbf.png">

## タイム＆上がり3F横
期間を指定してDataFrame型で出走タイムを比較できます。
<img width="855" alt="スクリーンショット 2023-04-20 0 29 22" src="https://user-images.githubusercontent.com/74707967/233125268-b2e7b1cd-1fab-4b2e-b554-37e04de45acc.png">

## パドック
過去のパドックの動画を見ることができるURLを出力します。
(各地方競馬の公式YouTubeの時間指定付きのリンクが出力されます)
<img width="845" alt="スクリーンショット 2023-04-20 0 40 00" src="https://user-images.githubusercontent.com/74707967/233128188-e1e886df-49c0-49c8-92fe-97575fe235b1.png">

過去の対戦成績を見ることができます。
